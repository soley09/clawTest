## 功能介绍

clawTest 是一款用于处理Word文档的工具，主要功能包括：

### 1. Word校对
- 读取Word文档
- 根据知识库进行错别字校对
- 生成校对后的新文件

### 2. 知识库管理
- 管理专业术语
- 管理错词-正确词对应表
- 生成候选错词
- 批量导入词库

### 3. 精简Word
- 将同一人连续说的多段话合并成一段
- 适用于心理催眠对话记录的整理

## 安装配置

### 环境要求
- Windows 7/8/10/11
- Python 3.8 或更高版本

### 安装步骤

#### 第一步：安装Python
1. 访问 Python 官网：https://www.python.org/downloads/
2. 下载 Python 3.10 或 3.11 版本
3. 运行安装程序
4. **重要：** 勾选 "Add Python to PATH"
5. 点击 "Install Now"

#### 第二步：安装依赖库
1. 双击运行 `install.bat`
2. 脚本会自动检测并安装 python-docx 依赖

#### 第三步：运行程序
双击以下文件：
- `启动Word校对.bat` - 打开Word校对工具（推荐）
- `启动知识库管理器.bat` - 打开知识库管理器
- `run.bat` - 菜单选择程序

## 使用方法

### 启动方式

#### 方式一：直接启动（推荐）
- 双击 `启动Word校对.bat` - 打开Word校对工具
- 双击 `启动知识库管理器.bat` - 打开知识库管理器

#### 方式二：菜单选择
1. 双击 `run.bat`
2. 输入数字选择要运行的程序

---

### Word校对工具使用方法

1. 点击 "📂 选择要被校准的文件" 按钮
2. 选择要校对的Word文档（.docx或.doc）
3. 文档内容会显示在左侧区域
4. 点击 "🔧 执行校准" 按钮
5. 校对后的内容会显示在右侧
6. 点击 "💾 导出" 保存结果

### 精简Word使用方法

1. 在Word校对工具界面右侧
2. 点击 "📂 选择被精简的Word"
3. 选择要精简的文档
4. 点击 "✂️ 执行精简"
5. 会在同目录生成 "xxx_精简.docx"

**精简规则：**
- 将同一人连续说的多段话合并成一段
- 适用于心理催眠对话记录

### 知识库管理器使用方法

#### 管理专业术语
1. 切换到 "术语" 标签
2. 点击 "➕ 添加专业术语" 添加新术语
3. 支持批量添加（用空格、逗号分隔）
4. 可以在 "待审核" 标签审核生成的错词候选

#### 管理词库
1. 切换到 "词库" 标签
2. 查看已导入的错词-正确词对应
3. 支持按添加顺序或字母顺序排序

## Python路径配置

### 自动查找
程序会自动按以下顺序查找Python：
1. `D:\Miniconda\python.exe`
2. `D:\Miniconda3\python.exe`
3. `config.bat` 中配置的路径
4. 系统PATH中的python（排除Windows Apps）

### 手动配置
如自动查找失败，可编辑 `config.bat`：

```batch
set PYTHON_PATH=C:\Python\python.exe
```

将路径改为你实际的Python安装位置即可。

## 常见问题

### Q1: 运行程序时提示 "python-docx 库未安装"
**解决方法：** 双击运行 `install.bat` 安装依赖

### Q2: 双击bat文件提示找不到Python
**解决方法：**
1. 确保Python已安装
2. 或编辑 `config.bat` 配置正确的Python路径

### Q3: 精简Word没有效果
**解决方法：** 确保文档格式为 "名字(时间戳): 内容"，例如：
```
阿标(00:00:06): 第一句话
阿标(00:00:07): 第二句话
```

### Q4: 如何更新词库？
**解决方法：** 在知识库管理器中：
1. 导入新的错词列表
2. 或在 "待审核" 标签审核并导入候选错词

### Q5: 启动后界面闪现消失
**解决方法：** 确保使用 `启动Word校对.bat` 或 `启动知识库管理器.bat`，不要直接双击 `.py` 文件
