======================================
快速开始
======================================

1. 安装Python（如已安装请跳过）
   - 访问 https://www.python.org/downloads/
   - 下载并安装 Python 3.10 或更高版本
   - 安装时勾选 "Add Python to PATH"

2. 安装依赖
   - 双击运行 install.bat

3. 运行程序
   - 双击 "启动Word校对.bat" 打开Word校对工具
   - 双击 "启动知识库管理器.bat" 打开知识库管理器
   - 或双击 run.bat 选择要运行的程序

======================================
主要功能
======================================

1. Word校对 - 错别字校对
2. 知识库管理 - 管理专业术语和词库
3. 精简Word - 合并对话记录

======================================
启动方式
======================================

推荐使用以下启动方式（自动查找Python）：
- 启动Word校对.bat      - 打开Word校对工具
- 启动知识库管理器.bat  - 打开知识库管理器
- run.bat               - 菜单选择

如遇Python找不到的问题：
- 方案1: 确保Python已安装并添加到PATH
- 方案2: 编辑config.bat，配置PYTHON_PATH
